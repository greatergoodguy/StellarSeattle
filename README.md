# Stellar Seattle

This code base will run as is. Just clone it and run in Android Studio.

	* Search
	* Map
	* Venue Details
	* Favorite

	* Picasso	
	* MVVM
	* Databinding
3. Testing
	* Unit Test
	* ViewModel Test
		* Mockito
	* UI Test
